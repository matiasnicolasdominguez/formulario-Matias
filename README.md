# formulario-Matias

This is my form website where you can fill out forms to attend events or other things. I programmed this myself, and it works the same as Google Forms or Microsoft Forms, although it will also have some distinguishing features soon.

It also has some basic functions at the moment, but it even has functions in the code that are more or less inactive, which are related to functions such as being able to send your resume, obviously like in Google Forms, but also being able to send some images so there is more transparency, or even your voice function in the case of interviews.
